Sobre este aplicativo Python da Cloud Foundry

De onde vêm os dados?


As informações sobre as ações são provenientes da EOD US Stock Prices, publicadas pela QUOTEMEDIA gratuitamente; este aplicativo faz requisições utilizando a API do Quandl para capturar os dados em formato JSON. Posteriormente os dados são inseridos em um DataFrame Pandas para análise.


Como funciona a predição?


As predições são feitas utilizando a biblioteca fbprophet, que executa um algoritmo de aprendizado de máquina voltado para a modelagem e predição de conjuntos de dados do tipo TimeSeries. O fbprophet é baseado em modelos aditivos (regressão não-paramétrica).
